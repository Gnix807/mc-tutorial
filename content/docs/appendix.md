# 附录

## 参考来源

本教程在撰写过程中参考了以下资料：

-   [Minecraft Wiki（中文）](https://zh.minecraft.wiki/)——游戏机制、版本记录、合成配方、生物数据等
-   [MC百科](https://www.mcmod.cn/)——模组信息、社区教程
-   [《你缺失的那门计算机课》](https://www.criwits.top/missing/)——写作风格参考

本教程中所使用的游戏截图来源于 Minecraft 游戏内截图、Minecraft Wiki 以及 MC百科。所有截图中出现的 Minecraft 相关素材版权归 Mojang Studios / Microsoft 所有，仅用于教学与介绍目的。
