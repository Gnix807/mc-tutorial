# 2.1 第一天

> [!NOTE]
> **基于版本**：Java 版 26.2 · 生存模式 · 默认难度

几乎每个 Minecraft 老玩家都记得自己的第一个夜晚——缩在徒手挖出的土洞里，听着外面僵尸的呻吟和蜘蛛的脚步声，手里只有可怜的两三根火把。

这篇教程的目的，是让你不必重蹈这种「硬核体验」的覆辙。我们将从容地度过第一个昼夜，天亮时拥有一套完整的石器装备、充足的食物，以及一个安全的临时居所。

## 创建你的第一个世界

启动游戏，在主菜单依次点击：

1.  **单人游戏 → 创建新的世界**
2.  世界名称随便填，叫「生存之旅」或「MC 第一档」都可以
3.  游戏模式选**生存**
4.  难度保持默认的**普通**即可

{{< image src="/images/ch02/1.png" alt="创建世界" title="创建新世界" width="750" >}}

> [!TIP]
> 如果这是你第一次玩 Minecraft，建议用默认设置直接开始。等熟悉了基本操作之后，再去研究种子、超平坦这些高级选项也不迟。

## 认识你的 HUD

进入世界后，屏幕上贴着的各种信息统称为 **HUD（平视显示器）**。它们是你在游戏中最重要的信息来源。

{{< image src="/images/ch02/2.png" alt="HUD 界面" title="Minecraft 的 HUD 界面" width="750" >}}

### 生命、饥饿，以及那个你看不到的饱和度

Minecraft 中有三条与生存直接相关的数值：

| 数值 | 显示 | 上限 | 说明 |
|---|---|---|---|
| 生命值 | 红心 <img src="https://zh.minecraft.wiki/images/Heart_%28icon%29.png?16571" style="zoom:150%;" /> | 20 点（10 颗） | 归零即死亡 |
| 饥饿值 | 鸡腿 <img src="https://zh.minecraft.wiki/images/Hunger.svg?5c3ee" style="zoom:150%;" /> | 20 点（10 根） | 耗尽后持续扣血；低于 6 点时无法疾跑 |
| 饱和度 | 不可见 | ≤ 饥饿值 | 饥饿值的「缓冲层」，消耗完才会轮到饥饿值 |

> [!NOTE]
> 饱和度是一个隐藏数值，你可以把它理解为你吃下去但还没来得及完全消化的那部分。牛排比面包「撑得久」，是缘于它提供了更高的饱和度。当饱和度彻底耗尽时，饥饿值图标会轻微抖动——这是你唯一能察觉到的信号。

饱和度还有一个关键作用：它直接影响你回血的速度。

-   饥饿值满 + 有剩余饱和度 → 快速回血（每 0.5 秒最多 1 点）
-   饥饿值 ≥ 18 → 缓慢回血（每 4 秒 1 点）
-   饥饿值 < 18 → **不！会！回！血！**

所以战斗中受伤后，别光站着等——先吃高饱和度的食物填饱肚子，回血效率差别很大。

### 屏幕上的其他信息

{{< card-list columns="2" >}}
-   **快捷栏（屏幕下方 9 格）**
    你当前随身携带的物品。按 {{< key >}}1 ~ 9{{< /key >}} 切换，鼠标滚轮也可以。
-   **经验条（绿色进度条）**
    击杀生物、挖掘矿石、烧炼物品均可获得。经验用于附魔和修复，第一天可以先不管。
-   **十字准星**
    屏幕正中央。你对世界的所有交互——挖掘、攻击、放置、使用——都由它来瞄准。
    {{< /card-list >}}

## 几种实用的移动技巧

在开始冒险之前，先了解几个能让你跑得更快、更省力的技巧：

{{< card-list columns="2" >}}
-   **顶头跳**：在 2 格高的空间里疾跑并不断按跳跃键，角色会被天花板顶住，连续做出短促的跳跃。这种移动方式比平地疾跑更快，在矿道中赶路非常实用。不过它会大幅加速饥饿值的消耗。
-   **疾跑中攻击可节省食物**：疾跑消耗饥饿值很快，但如果在疾跑过程中攻击任意生物，「这段疾跑的消耗累积」会被重置。赶长路时顺手拍一下路过的动物，能省下不少粮食。
-   **从 4 格高度开始摔伤**：前 3 格以内安全落地，第 4 格起每多 1 格扣 1 点血。疾跑状态下最远可跳 4 格。
{{< /card-list >}}

## 你在什么环境里出生？

不同生物群系的资源丰富程度差异很大。根据出生点周围的景色，大致可以分为三种情况：

{{< card-list columns="3" >}}
-   **森林 / 针叶林 / 丛林等**
    树木茂密，开局最舒适的类型。直接撸树，一路顺畅。
-   **平原 / 草原 / 热带草原等**
    树木稀疏但动物众多。多走几步就能找到树，顺便打猎囤肉。
-   **沙漠 / 恶地 / 冰原等**
    几乎找不到树木。留意河谷中的零星树木，或者寻找开着粉色花朵的杜鹃树（它下方一定连接着繁茂洞穴）。如果附近有沙漠村庄，直接利用村庄资源也是上策。什么都没有的话……建议果断跑图，移动到更适合生存的群系。
    {{< /card-list >}}

## 第一步：撸树（0 ~ 2 分钟）

Minecraft 有一条广为流传的格言：**要想富，先撸树**。进入世界后的第一件事，就是找到离你最近的那棵树。

1.  走到树旁，将十字准星对准树干。
2.  按住**鼠标左键**不放，徒手敲击树干。
3.  树干碎裂后掉出一块原木。走过去即可自动拾取。
4.  一棵树通常掉落 4 ~ 6 块原木。建议砍完整棵，收集 **6 ~ 8 块**。

> [!NOTE]
> 徒手撸树速度不快。另外，只有树干才会掉落原木——树叶不会，而且没有工具的话破坏树叶也相当费时。

## 第二步：打造第一批工具（2 ~ 5 分钟）

按下 {{< key >}}E{{< /key >}} 打开物品栏。右上角的 2×2 小方格就是你的合成栏，合成栏底部的绿色书本是**配方书**——它会列出你当前能够合成的所有物品。

{{< image src="/images/ch02/3.png" alt="物品栏界面" title="物品栏与合成栏" width="750" >}}

### 从原木到木板

把原木放进合成栏的任意一格，右侧就会产出 4 个木板。1 原木 = 4 木板。先攒出 **24 个左右**的木板备用。

{{< image src="/images/ch02/4.png" alt="原木合成木板" title="原木合成木板" width="750" >}}

### 制作工作台

用 4 块木板填满 2×2 合成栏，得到一个**工作台**。把它放到快捷栏，找个空地右键放置。

{{< image src="/images/ch02/5.png" alt="合成工作台" title="合成工作台" width="750" >}}

### 第一批木制工具

右键打开工作台，它提供 3×3 的合成空间——能合成的物品一下子多了不少。

{{< image src="/images/ch02/6.png" alt="工作台界面" title="工作台界面" width="750" >}}

你需要制作以下物品：
-   **木棍**（工具手柄）：2 块木板上下排列 = 4 根木棍

{{< image src="/images/ch02/7.png" alt="合成木棍" title="合成木棍" width="750" >}}

-   **木镐**（用来挖石头）：3 块木板铺顶行，中间一列放 2 根木棍

{{< image src="/images/ch02/8.png" alt="合成木镐" title="合成木镐" width="750" >}}

> [!TIP]
> 配方书可以帮你自动摆放材料——点击你想要合成的物品，材料会自动填入正确的位置。不用死记配方。

## 第三步：进入石器时代（5 ~ 10 分钟）

木制工具的效率实在不高。我们需要尽快升级到石制工具。

### 寻找石头

在出生点附近寻找裸露的灰色岩石——山坡断面是最常见的来源，河流岸边或地面凹陷处也常有石头露出。如果实在找不到，可以在水平方向挖几格再往下——但**绝对不要竖直挖自己脚下的方块**。

{{< image src="/images/ch02/9.png" alt="寻找石头" title="露天石头" width="750" >}}

### 升级装备

用木镐挖掘至少 **13 块圆石**，然后在工作台中合成：

-   **石镐**（3 圆石 + 2 木棍）

{{< image src="/images/ch02/10.png" alt="合成石镐" title="合成石镐" width="750" >}}

-   **石剑**（2 圆石 + 1 木棍）

{{< image src="/images/ch02/11.png" alt="合成石剑" title="合成石剑" width="750" >}}

-   **熔炉**（8 块圆石围成圈——用来烧肉和矿石）

{{< image src="/images/ch02/12.png" alt="合成熔炉" title="合成熔炉" width="750" >}}

> [!WARNING]
> 木镐只能挖石头和煤矿。看到铁矿石、青金石等，用木镐去敲只会白白浪费耐久——什么也挖不下来。工具的材质决定了你能挖掘的矿石等级，这是 Minecraft 最基础的规则。

## 第四步：解决温饱问题（10 ~ 15 分钟）

你会发现饥饿值在慢慢往下掉。在太阳落山前，你需要搞定食物。

### 获取食物的三种途径

{{< card-list columns="3" >}}
-   **狩猎动物**
    攻击野外的猪、牛、羊、鸡，它们会掉落生肉。生肉放进熔炉烧熟后，恢复的饥饿值会翻好几倍。
-   **捡苹果**
    砍伐橡树或深色橡树时，树叶有概率掉落苹果。捡到就能直接吃。
-   **采集种子**
    破坏草地可获得小麦种子。今天来不及种了，先收着，等安顿下来再说。
    {{< /card-list >}}

### 烧制食物

放下熔炉，打开界面：**上层放生肉，下层放燃料**（木板、原木、木炭都可以）。稍等片刻，熟肉就出炉了。

{{< image src="/images/ch02/13.png" alt="熔炉烧肉" title="熔炉烧制食物" width="750" >}}

### 制作火把——你的指路明灯

火把不仅是光源，更是你**防止迷路**和**阻止怪物生成**的关键道具：

1.  在熔炉中将一块原木烧成**木炭**
2.  1 木炭 + 1 木棍（上下排列）= **4 根火把**

{{< image src="/images/ch02/14.png" alt="合成火把" title="合成火把" width="750" >}}

出发探索前，先做 **16 ~ 20 根**火把。沿途在显眼的位置插上火把，回头时顺着光走，永远不会迷路。

## 第五步：天黑前，建好你的避难所

太阳一落山，野外就会大量刷新僵尸、骷髅、蜘蛛和苦力怕。你需要在天空完全变橙色之前，找到一个能保护自己的空间。

### 紧急避难所（一分钟版本）

1.  找一面山壁或土坡
2.  水平挖入 3 ~ 5 格深、2 格高
3.  在洞内插一根火把
4.  用方块**封死洞口**

{{< image src="/images/ch02/15.png" alt="紧急避难所" title="紧急避难所示例" width="750" >}}

这就够了。一个能让你安全度过第一夜的微型避难所。

### 标准小屋（五分钟版本）

如果你有充裕的时间，可以建得更舒适一些：

{{< timeline >}}
{{< timeline-item "步骤 1：平整地面" >}}
找一块 5×5 左右的平坦地面，清除杂草。
{{< /timeline-item >}}
{{< timeline-item "步骤 2：砌墙" >}}
用木板或圆石围出四面墙壁，每面至少 3 格高。
{{< /timeline-item >}}
{{< timeline-item "步骤 3：封顶" >}}
用方块铺满天花板。蜘蛛能爬墙——没有屋顶的话它们会从天而降。
{{< /timeline-item >}}
{{< timeline-item "步骤 4：安装门" >}}
6 块木板 = 1 扇门。装在一面墙上，右键开合。
{{< /timeline-item >}}
{{< timeline-item "步骤 5：内部照明" >}}
插 2 ~ 3 根火把，确保屋内的每个角落都有充足光线。
{{< /timeline-item >}}
{{< /timeline >}}

{{< image src="/images/ch02/16.webp" alt="标准小屋" title="标准庇护所外观" width="750" >}}

### 庇护所的基本配置

一个好的第一天庇护所至少应该包括：

-   **门**：6 块木板合成。挡住外面的怪物，也给你一个进出世界的窗口
-   **工作台 + 熔炉**：放在屋里，晚上还能继续干活
-   **箱子**：8 块木板围成圈。清理杂物栏的必备家具
-   **床**（如果有材料的话）：3 块羊毛 + 3 块木板。右键睡觉即可跳过夜晚，同时设定重生点

> [!WARNING]
> 不要在墙壁上留 1×1 的空隙——小僵尸刚好能钻进来。也不要在门边留下可以让怪物站上去翻墙的方块。

## 第六步：在屋里度过第一夜（夜晚，约 7 分钟）

躲进庇护所后，你并不是无事可做：

{{< card-list columns="2" >}}
-   **烧制食物**：把白天收集的生肉放进熔炉，为明天备好干粮。
-   **整理物品**：将多余的物资存入箱子，腾出快捷栏空间。
-   **制作备用工具**：多备一把石镐和石剑，以防关键时刻工具损坏。
-   **规划明天的行动**：按 {{< key >}}F3{{< /key >}} 查看坐标并记下家的位置，或者观察家周围的地标特征。
{{< /card-list >}}

如果你用羊毛和木板合成了床，右键睡觉就能直接跳到清晨，同时把重生点设在这里。

没有床的话，就在屋里安心等到天亮。外面怪物的声音虽然吓人，但有墙和门挡着，它们是进不来的。

{{< image src="/images/ch02/17.png" alt="夜晚屋内" title="在庇护所里度过夜晚" width="750" >}}

> [!NOTE]
> **关于死亡**：如果生命值降为零，你的角色会死亡。所有携带的物品掉落在原地，而你会在出生点（或上一次睡过的床旁边）复活。你有大约 5 分钟的时间跑回死亡地点捡回物品。这也是第一天必须造庇护所的原因——活下去，你才有机会继续你的冒险。

## 第二天早晨

太阳升起后，打开门，确认门外没有残留的怪物（僵尸和骷髅会被阳光灼烧，但蜘蛛和苦力怕不会消失）。

第二天你应该做的事：

1.  **建立食物来源**：猎杀更多动物，或开始垦荒种植小麦
2.  **继续采矿**：寻找煤矿和铁矿，它们在地表的山坡和地下洞穴中很常见
3.  **标记你的家**：用火把、高柱子或显眼的方块在家周围做路标
4.  **扩建居所**：为工作台、熔炉和箱子腾出更宽敞的空间

## 第一天总结

{{< card-list columns="2" >}}
-   撸树 → 造木板 → 做工作台
-   造木镐 → 挖圆石 → 升级石器
-   打猎采集 → 烧熟肉 → 解决饥饿
-   木头变木炭 → 合成火把 → 照亮前路
-   砌墙 + 封顶 + 装门 → 安全度过夜晚
-   夜晚在屋里休整准备，天亮再出发探索
{{< /card-list >}}
