# 启动器与版本

> [!NOTE]
> **基于版本**：Java 版 26.x · 启动器说明覆盖官方和第三方 · 版本表截至 2026 年 7 月

购买游戏之后，你需要通过**启动器**来下载和运行 Minecraft。启动器不仅是游戏的入口，还负责版本管理、模组加载、资源包切换等核心功能。本节将介绍官方启动器以及国内主流的几款第三方启动器。

## 什么是启动器？

启动器（Launcher）是 Minecraft 的客户端管理程序，它的主要功能包括：

-   **下载与安装**：从 Mojang 服务器获取游戏文件，自动安装对应版本
-   **版本管理**：在同一台电脑上保留多个游戏版本，随时切换
-   **账号登录**：通过 Microsoft 账户认证，验证游戏许可
-   **游戏配置**：调节内存分配、JVM 参数、游戏分辨率等
-   **模组与整合包**：方便地安装 Forge、Fabric 等模组加载器，导入整合包

简单来说，启动器就是你进入方块世界的「大门」。

## 官方启动器

### Minecraft Launcher

Minecraft Launcher 是 Mojang 官方出品的启动器，也是购买游戏后最先接触的启动器。

{{< card-list columns="2" >}}

    最安全可靠，自动更新，支持 Java 版和基岩版一站式管理。

-   主打原版生存，不需要频繁切换版本和模组

-   需要同时管理 Java 版和基岩版

### 下载与安装

1.  访问 [Minecraft 下载页面](https://www.minecraft.net/download) 。
2.  根据你的操作系统选择对应版本，下载安装程序。
3.  运行安装程序，按照提示完成安装。
4.  启动后登录你的 Microsoft 账户，即可在左侧看到已购买的游戏版本。

初次启动时，启动器会自动下载对应版本的 Minecraft 游戏文件。这个过程可能需要几分钟，具体时间取决于网络速度。

## 第三方启动器

对于想要深入游玩 Minecraft 的玩家来说，第三方启动器几乎是必选项。它们提供了更丰富的版本管理、模组安装和个性化功能。下面介绍国内最主流的几款。

> [!NOTE]
> **所有第三方启动器都需要正版账号才能登录游玩**。它们不会帮你绕过正版验证，只是提供了比官方启动器更好的使用体验。

{{< badge text="最推荐" >}} 国内使用人数最多的第三方启动器，全名 **Plain Craft Launcher 2**，由个人开发者「龙腾猫跃」独立开发并持续维护，完全免费。

-   **全中文体验**
    崩溃报告、Mod 冲突提示、Java 路径全部是中文，不用猜哪里出了问题。
-   **功能全面**
    一键安装 Forge / Fabric / OptiFine 等模组加载器，支持整合包导入，内置帮助系统。

**下载方式**：在 [PCL2 官方爱发电页面](https://afdian.com/a/LTCat) 或 [GitHub 仓库](https://github.com/Hex-Dragon/PCL2) 免费获取。下载后双击 `.exe` 文件即可启动，无需安装。首次启动会自动更新到最新版本。官方提供了 [使用说明文档](https://shimo.im/docs/qKPttVvXKqPD8YDC)。

> [!TIP]
> **我该玩哪个版本？**
> -   纯原版体验 → **最新正式版**（目前是 26.2）
> -   大型模组整合包 → **1.20.1**（Fabric 生态最成熟）或 **1.16.5**（Forge 生态最成熟）
> -   经典老牌整合包（GTNH 等）→ **1.7.10 / 1.12.2**
> -   PvP 小游戏 → **1.8.9**（1.9 战斗机制改版前的经典版本）

### 版本类型

在启动器的版本列表中，你会看到不同类型的版本标识：

{{< card-list columns="3" >}}
-   **正式版（Release）**
    经过充分测试的稳定版本，推荐所有玩家使用。没有任何特殊的后缀标识。
-   **快照版（Snapshot）**
    开发中的预览版本，包含新特性但可能不稳定。版本号类似 `24w33a`。
-   **预发布版（Pre-release）**
    正式版发布前的候选版本，稳定性介于快照和正式版之间。
-   **远古版（Old Alpha/Beta）**
    2010-2011 年的古老测试版本，怀旧玩家专用。
    {{< /card-list >}}

> [!WARNING]
> **新手提示**：安装启动器后，直接选择最新的**正式版**即可。快照版和预发布版可能存在 Bug，不建议作为主力游玩版本。后期玩模组时需要选择特定旧版本，届时启动器会自动下载对应版本文件。
