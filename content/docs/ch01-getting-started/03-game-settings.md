# 游戏设置

> [!NOTE]
> **基于版本**：Java 版 26.1 ~ 26.2 · 选项界面在 25w41a（1.21.11 快照）后有过较大调整，本教程反映最新布局 · 按键绑定和调试快捷键适用于 Java 版

安装好启动器、下载完游戏之后，在开始冒险之前，建议先花几分钟调整一下游戏设置。合适的设置能大幅提升游戏体验——画面更流畅、操作更顺手、声音更舒适。

本节内容参考 [Minecraft Wiki](https://zh.minecraft.wiki/w/%E9%80%89%E9%A1%B9) 编写，聚焦于新手最需要关注的选项。为照顾纯正的新人，本节已大幅简化了内容，如对设置内容感兴趣，欢迎前往维基查看。

## 打开设置菜单

进入游戏主菜单后，点击「**选项…**」即可进入设置界面。主要包含以下几个子菜单：

{{< card-list columns="2" >}}
-   **视频设置**
    画面品质、帧率、渲染距离、全屏等
-   **音乐和声音**
    主音量、各类音效独立调节
-   **按键控制**
    键盘和鼠标按键绑定
-   **语言**
    游戏界面显示语言
-   **辅助功能设置**
    字幕、高对比度、视角摇晃等
-   **聊天设置**
    聊天显示方式和样式
    {{< /card-list >}}

## 视频设置

视频设置直接影响游戏的画面质量和流畅度，是最重要的一组选项。

### 显示类

{{< badge text="关键设置" >}}

| 选项 | 推荐值 | 说明 |
|---|---|---|
| **全屏** | {{< badge text="开" >}} | 获得沉浸式体验，按 `F11` 快速切换 |
| **界面尺寸** | 自动 | 数值越大 UI 越大，高分辨率屏幕可适当调大 |
| **亮度** | 50 ~ 100 | 调高可以让洞穴和夜间更亮，纯视觉效果 |
| **最大帧率** | 120fps 或无限制 | 性能足够就选无限制，笔记本建议限制到 60~120 |
| **垂直同步** | {{< badge text="开" >}} | 消除画面撕裂，但可能增加延迟 |

### 品质与性能

{{< badge text="重点" >}} 这组选项直接影响游戏帧率。

| 选项 | 推荐值 | 说明 |
|---|---|---|
| **渲染距离** | 12~16 区块 | 越大看得越远，但更吃性能。中低配电脑建议 12 |
| **模拟距离** | 8~12 区块 | 影响实体更新范围，对性能影响比渲染距离更大 |
| **平滑光照** | {{< badge text="开" >}} | 光照过渡更自然，性能影响很小 |
| **云** | 流畅/高品质 | 「高品质」是立体云，「流畅」是平面，性能差别不大 |
| **粒子效果** | 全部/减少 | 中低配建议「减少」，不影响游戏玩法 |
| **实体阴影** | {{< badge text="开" >}} | 生物脚下出现阴影，画面更真实 |
| **实体渲染距离** | 100% | 控制生物在多远可见，调低可提升性能 |
| **可透视树叶** | {{< badge text="开" >}} | 树叶透明显示，方便在树林中观察 |
| **区块构建器** | 线程化 | 减少放置方块时的卡顿，推荐 |

> [!TIP]
> **中低配电脑优化方案**：渲染距离 8~10、模拟距离 8、粒子效果「减少」、实体渲染距离 75%。这几个选项对帧率影响最大。

### 视场角（FOV）

视场角控制画面的视野范围。默认值是 **70**，这是一个比较舒适的值。

-   **调低（50~60）**：画面拉近，适合 PvP 或精细操作
-   **调高（80~90）**：画面更广，视野开阔，但边缘会略微变形
-   **「广角」（110）**：极限视野，类似运动相机效果

一般玩家选 **70~80** 即可。

## 音乐和声音

Minecraft 的音乐和音效是其氛围感的核心。建议按照以下方式调整：

| 选项 | 推荐 | 说明 |
|---|---|---|
| 主音量 | 80% | 保留一些系统声音空间 |
| 音乐 | 60~80% | 背景音乐音量 |
| 友好生物 | 80% | 动物、村民等 |
| 敌对生物 | {{< badge text="100%" >}} | 听到嘶嘶声能救命！ |
| 方块 | 60~80% | 挖掘、放置等音效 |
| 环境 | 60~80% | 水声、矿车等 |
| 玩家 | 80% | 受伤、掉落等反馈 |
| **隐藏式字幕** | {{< badge text="开" >}} | 以文字显示声音方向和来源，强烈推荐 |

> [!NOTE]
> 打开「**隐藏式字幕**」后，屏幕右下角会以文字形式显示周围的声音（例如「<僵尸：低吼 」「脚步声>」）。这是新手的神器——能帮你提前发现威胁，也方便听障玩家游玩。

另外，打开「**音乐弹窗**」可以在歌曲切换时在左上角看到曲目名称，方便收藏喜欢的音乐。

## 按键控制

Minecraft 的按键控制分为「移动」「杂项」「多人游戏」「游戏内容」「物品栏」「创造模式」「旁观模式」「调试」等类别。下面按使用频率整理。

### 移动与基础操作

{{< card-list columns="3" >}}
-   {{< key >}}W A S D{{< /key >}}
    前后左右移动
-   {{< key >}}空格{{< /key >}}
    跳跃 / 双击飞行
-   {{< key >}}左 Shift{{< /key >}}
    潜行（搭桥必备）
-   {{< key >}}左 Ctrl{{< /key >}}
    疾跑 / 速泳
-   {{< key >}}鼠标左键{{< /key >}}
    攻击 / 长按挖掘
-   {{< key >}}鼠标右键{{< /key >}}
    使用物品（部分物品需长按使用） / 放置方块
-   {{< key >}}鼠标中键{{< /key >}}
    选取方块
-   {{< key >}}鼠标滚轮{{< /key >}}
    切换快捷栏物品
-   {{< key >}}F5{{< /key >}}
    切换第一/第三人称视角

### 物品栏与快捷栏

| 按键 | 功能 |
|---|---|
| {{< key >}}E{{< /key >}} | 打开 / 关闭物品栏 |
| {{< key >}}1 ~ 9{{< /key >}} | 选择快捷栏对应槽位 |
| {{< key >}}F{{< /key >}} | 交换主手和副手物品（火把 / 盾牌放副手） |
| {{< key >}}Q{{< /key >}} | 丢弃 1 个物品 |
| {{< key >}}Ctrl + Q{{< /key >}} | 丢弃**整组**物品 |
| {{< key >}}鼠标中键{{< /key >}} | **选取方块**：准星对准方块，自动切换到对应物品 |
| {{< key >}}鼠标左键双击{{< /key >}} | 整理散落物品为一叠 |
| {{< key >}}鼠标左键拖拽{{< /key >}} | 均分一叠物品到多个槽位 |
| {{< key >}}鼠标右键拖拽{{< /key >}} | 在每个经过的槽位放 1 个物品 |
| {{< key >}}Shift + 点击{{< /key >}} | 快速转移物品（背包 ↔ 箱子 / 工作台 / 熔炉） |

> [!TIP]
> **Shift + 点击**是最常用的物品栏快捷键。无论是把矿放入熔炉、把材料放入工作台、还是整理箱子库存，熟练使用 Shift + 点击可以节省大量反复拖拽的时间。

### 调试快捷键（F3 组合）

按下 {{< key >}}F3{{< /key >}} 打开调试屏幕后，配合以下按键实现各种功能：

| 组合键 | 功能 |
|---|---|
| {{< key >}}F3{{< /key >}} | 切换调试屏幕（显示坐标、帧率、生物群系等） |
| {{< key >}}F3 + G{{< /key >}} | **显示区块边界**（找史莱姆区块、做刷怪塔必备） |
| {{< key >}}F3 + H{{< /key >}} | **显示物品高级提示框**（查看耐久度、物品 ID） |
| {{< key >}}F3 + B{{< /key >}} | 显示实体碰撞箱（适合研究刷怪机制） |
| {{< key >}}F3 + A{{< /key >}} | 重新加载所有区块（画面卡了就按这个） |
| {{< key >}}F3 + T{{< /key >}} | 重新加载资源包和纹理 |
| {{< key >}}F3 + P{{< /key >}} | 切换失去焦点时是否暂停（切窗口不暂停） |
| {{< key >}}F3 + F4{{< /key >}} | 游戏模式切换器（创造 / 生存快速切换，需管理员权限） |
| {{< key >}}F3 + D{{< /key >}} | 清除聊天记录 |
| {{< key >}}F3 + C{{< /key >}} | 复制当前坐标（形如 `/execute in ... run tp x y z`） |
| {{< key >}}F3 + I{{< /key >}} | 复制准星处方块的数据到剪贴板 |
| {{< key >}}F3 + Esc{{< /key >}} | 按住 10 秒可手动触发崩溃（调试用） |

> [!NOTE]
> 在笔记本上，F3 可能需要配合 {{< key >}}Fn{{< /key >}} 键使用。如果你按 F3 没反应，尝试按住 {{< key >}}Fn + F3{{< /key >}}。也可以在系统设置中将功能键设为默认。

### 建议修改的按键

| 选项 | 推荐值 | 说明 |
|---|---|---|
| **潜行** | {{< badge text="切换" >}} | 搭桥时不用一直按着 Shift |
| **疾跑** | 按住（保持默认） | 方便临时加速 |

### 新手优先记住的快捷键

如果你是第一次玩 Minecraft，不需要一下子记住所有快捷键。建议先把下面这 **8 个** 练熟：

1. {{< key >}}Shift{{< /key >}} 潜行——防止摔下悬崖
2. {{< key >}}E{{< /key >}} 物品栏——随时查看和管理物品
3. {{< key >}}鼠标中键{{< /key >}} 选取方块——瞄准就能切换，超方便
4. {{< key >}}Shift + 点击{{< /key >}} 快速转移——告别反复拖拽
5. {{< key >}}F{{< /key >}} 副手交换——左手火把照亮洞穴
6. {{< key >}}F3{{< /key >}} 调试屏幕——迷路了看 XYZ 坐标
7. {{< key >}}F3 + G{{< /key >}} 区块边界——盖建筑做刷怪塔都用得上
8. {{< key >}}F5{{< /key >}} 切换视角——第三人称截图更好看

后续随着游戏深入，再慢慢掌握更多快捷键。

## 语言

在「语言」菜单中选择「**简体中文**」即可。如果需要更好的中文字体渲染，可以在「字体设置」中：

-   **强制使用 Unicode 字体**：建议保持关闭，使用默认字体即可正常显示中文
-   **日本字形变体**：保持默认

> [!WARNING]
> 如果安装了一些旧版本的模组或资源包，可能会出现中文乱码。这通常是字体文件不完整，不是游戏本体的问题。后续章节会介绍如何解决。

## 辅助功能设置

这组选项可以改善游戏的可玩性和舒适度：

| 选项 | 推荐 | 说明 |
|---|---|---|
| **视角摇晃** | {{< badge text="开" >}} | 走路时画面自然晃动，关了更流畅 |
| **黑暗脉动效果** | 降低到 50% | 减弱监守者给予的黑暗特效强度 |
| **附魔光效闪烁强度** | 50~75% | 默认的附魔光泽可能过亮 |
| **高对比度** | {{< badge text="关" >}} | 一般不需要 |
| **黑白徽标** | 关 | Mojang 加载界面的背景颜色 |

> [!TIP]
> 如果晕 3D 或者在洞穴中感到不适，可以尝试：关闭「**视角摇晃**」、降低「**视场角效果**」、调高「**亮度**」到 100。
