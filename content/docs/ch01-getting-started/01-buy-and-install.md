# 1.1 购买与安装

> [!NOTE]
> **基于版本**：Java 版 26.x / 基岩版 · 本节内容适用于所有版本。

在开始游戏之前，首先需要获得 Minecraft 的正版账号。本节将介绍 Minecraft 国际版的版本区别，以及通过官网和微软商店两种方式购买与安装游戏。

## Minecraft 国际版是什么？

Minecraft 目前主要分为两个版本体系：

{{< card-list columns="2" >}}
-   **Java 版** {{< badge text="推荐" >}}
    最早的 Minecraft 版本，使用 Java 语言编写。模组生态丰富、社区活跃，支持 Windows / macOS / Linux。本教程的主要讲解对象。
-   **基岩版**
    使用 C++ 重写，运行效率更高，跨平台联机方便（Windows / 主机 / 手机），但模组支持相对有限。
    {{< /card-list >}}

两个版本各自需要**单独购买**，互不包含。如果你主要用电脑玩，并且对模组、红石、命令等深度内容感兴趣，**推荐优先购买 Java 版**。

> [!NOTE]
> **什么是「国际版」？**
> 由于历史原因，Minecraft 在中国大陆由网易代理运营{{< tip "免费但更新慢，和国际版数据不互通" >}}我的世界中国版{{< /tip >}}，与国际版是**两个不同的游戏**。本教程所述内容全部基于**国际版**，请购买时注意区分。

## 账号体系说明

自 2021 年起，Mojang 账号已全面迁移至 {{< badge text="Microsoft 账户" >}}。现在购买 Minecraft 需要拥有一个 Microsoft 账户（如果没有，可以在购买过程中免费注册）。

一个 Microsoft 账户可以同时拥有 Java 版和基岩版的许可，且可以跨设备登录。

{{< badge text="推荐方式" >}} 最简单直接的购买渠道。

{{< timeline-item "步骤 3" >}}

> 所以在 PC 上购买时，你不需要纠结「到底买 Java 版还是基岩版」——你买到的是两个版本的合集。

## 安装游戏

购买完成后，安装非常简单：

### 通过官方启动器安装

1.  访问 [Minecraft 下载页面](https://www.minecraft.net/download) ，下载 {{< badge text="Minecraft Launcher" >}}（官方启动器）。
2.  运行安装程序，按照提示完成安装。
3.  启动 Minecraft Launcher，登录你的 Microsoft 账户。
4.  在启动器中，你会在左侧看到「Minecraft: Java Edition」和「Minecraft: Bedrock Edition」两个选项。
    {{< image src="/images/ch01/6.png" alt="启动器页面" title="启动器页面" width="700" >}}
5.  点击想要游玩的版本，启动器会自动下载并安装对应游戏文件。

> [!NOTE]
> **启动器是什么？**
> Minecraft Launcher（启动器）是游戏的入口程序，负责管理版本、下载更新、启动游戏。官方启动器一次安装后，可以同时管理 Java 版和基岩版。本教程后续章节也会介绍第三方启动器的使用。

## 常见问题

### 购买后能在几台电脑上玩？

Minecraft 的许可是跟**账号**绑定的，而不是跟设备。你可以在任意多台电脑上登录你的 Microsoft 账户并下载游玩。但{{< tip "除非你和朋友在同一局域网下联机" >}}**同一时间只能在一台设备上在线游玩**{{< /tip >}}。

### Java 版和基岩版的区别到底是什么？

| 项目 | {{< badge text="Java 版" >}} | {{< badge text="基岩版" >}} |
|---|---|---|
| 支持平台 | Windows / macOS / Linux | Windows / 主机 / 手机 |
| 模组支持 | 极丰富（Forge / Fabric） | 有限（Add-Ons） |
| 红石机制 | 准确定时、BUD、QC 等 | 有差异，部分机制不同 |
| 跨平台联机 | Java 版之间联机 | 基岩版之间跨平台联机 |
| 性能 | 一般，依赖 Java 优化 | 较好，C++ 原生 |

如果你主要玩原版生存和建造，选择哪个都可以。但如果你对模组、红石、数据包等深度玩法感兴趣，**Java 版是更好的选择**。

### 如何确认购买成功？

购买后可以访问 [Minecraft 官网](https://www.minecraft.net) 登录你的 Microsoft 账户，在「档案」中查看已购买的游戏许可。
