    1.2 启动器与版本#

    基于版本：Java 版 26.x · 启动器说明覆盖官方和第三方 · 版本表截至 2026 年 7 月

    购买游戏之后，你需要通过启动器来下载和运行 Minecraft。启动器不仅是游戏的入口，还负责版本管理、模组加载、资源包切换等核心功能。本节将介绍官方启动器以及国内主流的几款第三方启动器。

    什么是启动器？#

    启动器（Launcher）是 Minecraft 的客户端管理程序，它的主要功能包括：

    • 下载与安装：从 Mojang 服务器获取游戏文件，自动安装对应版本
    • 版本管理：在同一台电脑上保留多个游戏版本，随时切换
    • 账号登录：通过 Microsoft 账户认证，验证游戏许可
    • 游戏配置：调节内存分配、JVM 参数、游戏分辨率等
    • 模组与整合包：方便地安装 Forge、Fabric 等模组加载器，导入整合包

    简单来说，启动器就是你进入方块世界的「大门」。

    官方启动器#

    Minecraft Launcher#

    Minecraft Launcher 是 Mojang 官方出品的启动器，也是购买游戏后最先接触的启动器。

    • 优点 官方出品 最安全可靠，自动更新，支持 Java 版和基岩版一站式管理。
    • 不足 功能相对简单，无法直接安装第三方模组加载器，界面不够灵活。

    官方启动器适合以下玩家：

    • 主打原版生存，不需要频繁切换版本和模组
    • 对安全性要求高，不想折腾第三方工具
    • 需要同时管理 Java 版和基岩版

    下载与安装#

    1. 访问 Minecraft 下载页面 。
    2. 根据你的操作系统选择对应版本，下载安装程序。
    3. 运行安装程序，按照提示完成安装。
    4. 启动后登录你的 Microsoft 账户，即可在左侧看到已购买的游戏版本。

    初次启动时，启动器会自动下载对应版本的 Minecraft 游戏文件。这个过程可能需要几分钟，具体时间取决于网络速度。

    第三方启动器#

    对于想要深入游玩 Minecraft 的玩家来说，第三方启动器几乎是必选项。它们提供了更丰富的版本管理、模组安装和个性化功能。下面介绍国内最主流的几款。

    所有第三方启动器都需要正版账号才能登录游玩。它们不会帮你绕过正版验证，只是提供了比官方启动器更好的使用体验。

    PCL2（Plain Craft Launcher 2）#

    最推荐 国内使用人数最多的第三方启动器，全名 Plain Craft Launcher 2，由个人开发者「龙腾猫跃」独立开发并持续维护，完全免费。

    与同类启动器相比，PCL2 的核心优势在于「速度 + 中文 + 精简」：

    • 下载速度快 多线程下载针对国内网络优化，10M 宽带能跑满。内置多个镜像源自动选最快。
    • 全中文体验 崩溃报告、Mod 冲突提示、Java 路径全部是中文，不用猜哪里出了问题。
    • 功能全面 一键安装 Forge / Fabric / OptiFine 等模组加载器，支持整合包导入，内置帮助系统。
    • 零安装无广告 下载 EXE 双击即用，所有功能可关闭，不附带任何广告或捆绑软件。

    下载方式：在 PCL2 官方爱发电页面 或 GitHub 仓库 免费获取。下载后双击 .exe 文件即可启动，无需安装。首次启动会自动更新到最新版本。官方提供了 使用说明文档 。

    用 PCL2 下载游戏版本#

    打开 PCL2 后，点击顶部的「下载」标签页，版本列表会显示所有可用的游戏版本。选择你想要的版本，点击「开始下载」，等待自动完成即可。PCL2 会自动处理 Java 运行环境、游戏文件和资源包的下载。

    Mod 和整合包的安装将在第六章统一介绍。

    HMCL（Hello Minecraft! Launcher）#

    开源老牌 历史最悠久的国产开源启动器，由 huanghongxun 于 2015 年创建。代码完全开放，十余年持续维护至今，社区活跃。

    • 跨平台 支持 Windows / macOS / Linux，三大操作系统通用。
    • 自动 Java 管理 HMCL 会自动检测和下载所需的 Java 版本，新手无需手动折腾 Java 环境——启动器帮你搞定一切。
    • 版本隔离 不同游戏版本的 Mod、资源包、存档可以完全分开，互不干扰。装 Mod 和整合包必备。
    • 内置模组下载 直接搜索和安装 CurseForge 与 Modrinth 上的模组，自动检测前置依赖。

    下载方式：在 HMCL 官网 或 GitHub 免费获取。官方还提供了详尽的 使用文档 。

    用 HMCL 下载游戏版本#

    点击左侧「下载」→「游戏」选项卡，版本列表会按从新到旧的顺序显示。选择你想要的版本，点击右下角「安装」，等待自动完成即可。

    安装完成后回到主界面，点击「开始游戏」即可进入游戏。

    BakaXL#

    颜值担当 主打 UI 美学的启动器，以日系二次元风格著称。

    • 视觉惊艳 Metro 风格界面，支持自定义主题、背景图和动画效果。
    • 操作流畅 高度优化的 UI 交互，启动速度极快。
    • 主题商店 内置主题市场，可以下载社区创作的美化主题。
    • 整合包友好 支持 CurseForge / Modrinth 整合包一键导入。

    下载方式：在 BakaXL 官网 获取。官方提供了 使用文档 。

    启动器对比#

    启动器 平台 界面 模组支持 适合人群
    PCL2 Windows 现代 Material Forge / Fabric / OptiFine 大多数玩家
    HMCL Win/Mac/Linux 简洁实用 Forge / Fabric / OptiFine 跨平台用户
    BakaXL Windows 二次元华丽 Forge / Fabric 外观党

    三款都是免费软件，你可以都下载下来试试，看哪个最顺手就用哪个。个人推荐新手从 PCL2 入手，功能全面且上手难度最低。

    版本体系说明#

    Minecraft 的版本号看起来复杂，但只要掌握规律就很简单。

    大版本号#

    Minecraft 采用 主版本.次版本 的命名方式，每个大版本都有对应的中文名称。以下是主要大版本一览（数据来源： Minecraft Wiki ）：

    26.1 小鬼当家 2026 年 3 月
    1.21.5 春意盎然 2025 年 3 月
    1.21.4 苍园觉醒 2024 年 12 月
    1.21 棘巧试炼 2024 年 6 月
    1.20 足迹与故事 2023 年 6 月
    1.19 荒野更新 2022 年 6 月
    1.18 洞穴与山崖（第二部） 2021 年 11 月
    1.17 洞穴与山崖（第一部） 2021 年 6 月
    1.16 下界更新 2020 年 6 月
    1.15 嗡嗡蜂群 2019 年 12 月

    Mojang 每年夏季发布一个大版本更新，中间穿插数次小版本迭代。自 2025 年底起，版本号跳至 26.x 系列，游戏进入了全新的开发周期。

    我该玩哪个版本？

    • 纯原版体验 → 最新正式版（目前是 26.2）
    • 大型模组整合包 → 1.20.1（Fabric 生态最成熟）或 1.16.5（Forge 生态最成熟）
    • 经典老牌整合包（GTNH 等）→ 1.7.10 / 1.12.2
    • PvP 小游戏 → 1.8.9（1.9 战斗机制改版前的经典版本）

    版本类型#

    在启动器的版本列表中，你会看到不同类型的版本标识：

    • 正式版（Release） 经过充分测试的稳定版本，推荐所有玩家使用。没有任何特殊的后缀标识。
    • 快照版（Snapshot） 开发中的预览版本，包含新特性但可能不稳定。版本号类似 24w33a。
    • 预发布版（Pre-release） 正式版发布前的候选版本，稳定性介于快照和正式版之间。
    • 远古版（Old Alpha/Beta） 2010-2011 年的古老测试版本，怀旧玩家专用。

    新手提示：安装启动器后，直接选择最新的正式版即可。快照版和预发布版可能存在 Bug，不建议作为主力游玩版本。后期玩模组时需要选择特定旧版本，届时启动器会自动下载对应版本文件。

    版本的典型用途#

    版本 主要场景
    最新正式版（26.2） 纯原版生存、体验最新内容
    1.20.1 Fabric 模组生态最成熟
    1.19.2 大量优质整合包
    1.16.5 下界更新相关模组、老牌整合包
    1.12.2 经典大型科技魔法模组包
    1.8.9 小游戏服务器、PvP 对战
    1.7.10 GTNH 等超大型经典整合包

    如何选择启动器与版本？#

    这里给出一个简单的决策流程：

    新手 + 原版
    推荐官方启动器，安装简单、安全可靠。选择最新正式版即可开始游戏。
    新手 + 想玩模组
    推荐 PCL2。界面友好，内置一键安装模组加载器，下载速度快。版本选择 1.20.1（Fabric）或 1.16.5（Forge）。
    Mac / Linux 用户
    推荐 HMCL，三大平台通用。版本选择和 Windows 平台一致。
    外观控 + 整合包
    推荐 BakaXL，界面最华丽，主题市场丰富，整合包支持完善。
