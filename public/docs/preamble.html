    序#

    Minecraft 简史#

    2009 年 5 月 17 日，瑞典程序员 Markus “Notch” Persson 发布了一个用 Java 编写的简陋方块游戏——这就是 Minecraft 最早的版本（Classic）。在随后的两年里，Notch 几乎以一己之力推动游戏经历了 Indev、Infdev、Alpha、Beta 多个开发阶段，一边写代码一边根据社区反馈不断调整方向。

    2011 年 11 月 18 日，Minecraft 1.0 正式版在 MineCon 上发布。此时延斯·伯根斯坦（Jeb）已接任开发负责人，游戏从一个人的业余项目变成了一支团队的持续事业。

    2014 年 11 月 6 日，微软以 25 亿美元收购 Mojang 及 Minecraft 全部资产，Notch 从 Mojang 离职。此后 Minecraft 进入了持续扩张的时代：基岩版覆盖了移动设备和游戏主机，Java 版保持着每年一次大更新的节奏，教育版走进了全球课堂。

    截至 2026 年 6 月，Minecraft 全球销量已突破 4 亿份，是历史上销量最高的电子游戏。从 C418 的舒缓钢琴到 Lena Raine 的空灵氛围，它的原声音乐也已成为一代玩家的集体记忆。

    为什么要写这份教程？#

    红石电路可以搭建出一台计算机，命令方块能创造全新的游戏机制，模组生态更是层出不穷。然而，新手面对这些内容时往往会感到迷茫：该从哪开始？Java 版和基岩版有什么区别？红石到底怎么玩？

    这份教程正是为解答这些问题而写。

    如何使用本教程#

    本教程按照由浅入深的顺序编排，每一章都建立在前一章的基础之上。

    • 如果你是完全的新手，建议从第一章开始按顺序阅读。
    • 如果你已经有一定基础，可以直接跳转到你感兴趣的章节。
    • 教程中穿插了大量截图与示例，强烈建议在阅读时打开游戏跟着操作。

    适用版本#

    本教程主要基于 Minecraft Java 版 编写。截止本教程撰写时，最新正式版为 26.2（2026 年 6 月发布）。

    • Java 版是本教程的核心讲解对象。教程中的操作步骤、按键对照、游戏机制等，除非特别说明，均以 Java 版为准。
    • 基岩版仅在部分章节涉及（如购买安装、联机方式、版本差异对比等），会单独标注。
    • 如果某部分内容依赖于特定版本（例如某个版本才加入的功能），会在相应位置注明。

    Minecraft 更新频繁，教程尽量跟进最新版本，但也可能滞后。如有发现，欢迎反馈。
